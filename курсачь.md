Программное обеспечение которое позволяет мониторить доступность разных ресурсы особенно актуальна в современных условиях существования бизнес, госудаоственных и частных сервисов ИТ-энтузиастов. Всем им нужно поддерживать стабильную работу разных модулей ИТ-инфраструктуры, а для этого нужно постоянно мониторить работу разных ресурсов. Своевременное выявление недоступности ресурсов позволяет минимизировать потери финансового и репутационного характера. Автоматизация мониторинга с использованием многопоточных скриптов снижает нагрузку на специалистов DevOps и системных администраторов, что делает разработку подобных инструментов актуальной задачей.

Целью моей курсовой работы являеться разработка программного модуля для автоматизированного многопоточной проверки доступности веб-ресурсов с визуальным отображением результатов.

Для достижения поставленной цели будет необходимо решить следующие задачи:

- Изучить принципы работы HTTP протокола и коды статусов ответов от сервера.
- Выбрать [библиотеку](https://sky.pro/wiki/python/python-kak-osuschestvit-ping-serverov-cherez-icmp/ "subprocess, ping3, pythonping") для сетевого взаимодействия и освоить ее функции для реализации проекта.
- Проанализировать механизм многопоточной обработки данных в Python.
- Реализовать загрузку и валидацию списка  URI из внешнего файла.
- Разработать ядро проверки доступности с использованиям пула потоков.
- Создать интерфейс вывода результатов с цветовой индекацией статусов
- Протестировать приложение и оценить прирост производительности при многопоточном режиме.

Объектом исследования являеться процесс контроля доступа к веб-ресурсам. Предмет исследования - это алгоритм и программная реализация многопоточного HTTP-пингера на языке Python.

В проуессе разработке будут применятся методы структурного и объектно-ориентированного программирования, многопоточная обработка через библиотеку concurrent.futures, модульное тестирование и сравнительный анализ времени выполнения. Программная реализация выполняется в среде VS Code на языке Python 3.14 с использованием библиотеки ping3 для работы с сетью и flat для создания красивого графического интерфейса.
